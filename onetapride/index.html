Simple ride-sharing for seniors and their families

OneTapRide makes it easy for seniors to request rides from Uber and Lyft without the complexity of using multiple apps. Caregivers set up common routes, and seniors can request rides with just a few taps.

For Seniors

‚ùì Troubleshooting

No Routes Showing

  • Make sure routes are created in Settings ‚Üí Manage Routes
  • Check that routes are turned "On" (green toggle switch)
  • Ensure both places in the route still exist

Address Not Found

  • Try using the search feature instead of typing addresses
  • Always use the "Validate Address" button
  • Check for typos in street names or numbers

Ride App Doesn't Open

  • Make sure Uber or Lyft apps are installed
  • Update the ride apps if they're outdated
  • If apps aren't installed, a web page will open instead
